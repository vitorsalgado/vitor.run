# vitor.run

React + TypeScript + Vite site with a markdown-based blog.

## Blog

- **Content**: Add `.md` files in `src/content/posts/` with YAML frontmatter: `title`, `date`, and optional `description`.
- **Routes**: `/blog` lists all posts; `/blog/:slug` shows a post (slug = filename without `.md`).
- **SEO**: Each page sets document title, meta description, and canonical URL. Blog posts are loaded at build time (no runtime fetch), so the app works on any static host.

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run preview` — serve `dist/` locally
- `npm run e2e:blog` — build, start preview, and validate `/`, `/blog`, and `/blog/hello-world` return 200 (requires network for the preview server)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

